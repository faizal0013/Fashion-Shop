# Fashion Shop

This is simple **backend** Project

---

## Css Framework

- Tailwind Css

## Configuration

1. Change .env.temp file into .env file.
2. Change mongodb-username into your MongoDB atlas user-name.
3. Change mongodb-password into your MongoDB atlas user-password.

## Command

```
npm install
```

_or_

```
npm i
```

- _nodemon_

```
npm run nodemon
```

- _tailwind_

```
npm run tailwind
```
